例题：讨论\(f(x,y)\)在\((0,0)\)的可微性与偏导数的连续性:  

\(\displaystyle f(x,y)= \begin{cases} \frac{xy}{(x^2+y^2)^\alpha} & x^2+y^2 \neq 0 \\0 & x^2+y^2 = 0\end{cases}\)  

---

对\(f(x,y)\)的可微性考察:  

$$\displaystyle \lim_{(x,y) \to (0,0)} \frac{f(x,y) - [f(0,0) + f_x(0,0)x + f_y(0,0)y]}{\sqrt{x^2+y^2}} =\lim_{(x,y) \to (0,0)} \frac{xy}{(x^2+y^2)^{\alpha + \frac{1}{2}}} \stackrel{?}{=} 0$$

<br> 

- 当\(\alpha>\frac{1}{2}\)时,\((x^2+y^2)^{\alpha+\frac{1}{2}}\) 是 \(xy\) 的高阶无穷小,则\(f(x,y)\to\infty \ \ ((x,y)\to (0,0))\),则不可微  

- 当\(\alpha=\frac{1}{2}\)时,取\(y=kx\),则\( f(x,y)=\frac{xy}{x^2+y^2}=\frac{k}{1+k^2}\),依赖于k不可微  

- 当\(\alpha<\frac{1}{2}\)时,\(xy\) 是 \((x^2+y^2)^{\alpha+\frac{1}{2}}\)  的高阶无穷小,则\(f(x,y)\to0 \ \ ((x,y)\to (0,0))\),则可微  

---

对\(f_x(x,y)\)的连续性的考察(\(x\)与\(y\)对称)

\(\displaystyle f_x(x,y)=\frac{y[y^2+(1-2\alpha)x^2]}{(x^2+y^2)^{\alpha+1}}\)  

\(f_x(0,0)=\lim\limits_{x\to0}f_x(x,0)=0\) 

\
则考察:
\(\displaystyle \lim\limits_{(x,y)\to (0,0)}\frac{y[y^2+(1-2\alpha)x^2]}{(x^2+y^2)^{\alpha+1}}\stackrel{?}{=}0\)

<br>

- 当\((2\alpha+1)>3\)时,即\(\alpha>\frac{1}{2}\)时,\((x^2+y^2)^{\alpha+1}\) 是 \(y[y^2+(1-2\alpha)x^2]\) 的高阶无穷小,则\(f_x(x,y)\to\infty \ \ ((x,y)\to (0,0))\),则不连续  

- 当\((2\alpha+1)=3\)时,即\(\alpha=\frac{1}{2}\)时,取\(y=kx\),则\( f_x(x,y)=\frac{y[y^2+(1-2\alpha)x^2]}{(x^2+y^2)^{\alpha+1}}=\frac{k^3}{(1+k^2)^\frac{3}{2}}\),依赖于k不连续  

- 当\((2\alpha+1)<3\)时,即\(\alpha<\frac{1}{2}\)时,\(y[y^2+(1-2\alpha)x^2]\) 是 \((x^2+y^2)^{\alpha+1}\) 的高阶无穷小,则\(f_x(x,y)\to0 \ \ ((x,y)\to (0,0))\),则连续
