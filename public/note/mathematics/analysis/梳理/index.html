集合与映射
#

集合
#

  • 集合与运算
  • 实数集完备性
  • 确界存在原理

映射
#

  • 单射,满射,一一映射
  • 复合映射(复合函数)
  • 逆映射(反函数):
    • 反函数存在\(\iff\)函数是一一映射
    • 函数严格单调\(\implies\)反函数存在

极限
#

极限
#

  • 定义:\(\epsilon-N\)语言,\(\epsilon-M\)语言,\(\epsilon-\delta\)语言
  • 极限的性质:
    • 唯一性
    • (局部)有界性
    • (局部)保号性
    • (局部)保序性
    • 迫敛性
    • 四则运算
  • 数列极限与函数极限通过海涅归并原理相联系
  • 数列收敛准则相关:
    • 单调有界准则
    • 归并原理(拉链定理)
    • 闭区间套定理
    • Weierstrass定理
    • Cauchy收敛准则
    • *Stolz定理
  • 函数收敛准则相关：
    • 单调有界准则
    • Cauchy收敛准则

无穷小量(大量略)
#

  • 定义：
    • 无穷小:函数\(\alpha(x)\)满足\(\lim_{x\to x_0}\alpha(x)=0\)
    • 高阶无穷小,等价无穷小
  • 性质:
    • 有限个无穷小和/积为无穷小
    • 无穷小与有界函数的乘积为无穷小
    • \(\alpha(x)\sim\beta(x)\),则 \(\alpha(x)=\beta(x)+o[\beta(x)]\)

连续
#

连续
#

  • 定义:\(\forall \epsilon>0,\exists \delta>0 ,st. \forall |x-x_0|<\delta\)时有\(|f(x)-f(x_0)|<\epsilon\)
  • 连续性:
    • 连续函数的四则运算
    • 复合函数的连续性
    • 反函数的连续性
  • 闭区间上连续函数性质:
    • 有界性
    • 最值存在
    • 零点存在定理
    • 介质定理
    • 值域定理(值域为闭区间)

一致连续
#

  • 定义(非正式):连续函数在区间\(I\)上的\(\delta\)不依赖于\(x_0\)
  • 闭区间上连续\(\implies\)区间上一致连续

导数与微分
#

导数
#

  • 定义:导数(左右导数),可导,导函数
  • 函数可导\(\implies\)函数在区间上连续
  • 法则:
    • 求导基本法则
    • 四则运算
    • 复合函数求导
    • 反函数求导
    • 参数方程求导
  • 莱布尼兹公式:
    \((uv)^{(n)} = \sum_{k=0}^{n} \binom{n}{k} u^{(n-k)} v^{(k)}\)
  • 导函数单侧极限与函数单侧导数:
    若\(\lim_{x\to x_0+}f'(x)\)存在,则\(f'_{+}(x_0)\)存在,且二者相等

微分
#

  • 定义:\(\Delta y=\alpha \Delta x+o(\Delta x)\),\(\alpha \Delta x\)是线性主部
  • 在一元函数语境下,可微\(\iff\)可导
  • 一阶微分具有形式不变性,高阶不具备
  • 微分中值定理相关:
    • 导函数介质定理
    • Fermat定理
    • 罗尔中值定理
    • 拉格朗日中值
    • 柯西中值
  • 洛必达法则:
    • \(\frac{0}{0}\)型
    • \(\frac{\infty}{\infty}\)型
  • 泰勒定理:
    • Peano余项
    • Lagrange余项
    • Maclanrin公式
  • *Lagrange插值与Hermite插值

函数性态
#

  • 单调性
  • 极值
    \(f'(x)=0,f''(x)\neq0\)
  • 最值:极值与端点值中选取
  • 凹凸性与拐点
    • 凹凸性\(\iff\)曲弦关系\(\iff\)曲切关系

积分
#

定积分
#

  • 定义:划分+面积+求和+极限
  • 黎曼可积充要条件:
    • \(| \sum_{i=1}^{n} f(\xi_i) \Delta x_i - I | < \epsilon\)
    • \(\forall \epsilon > 0, \exists P \text{ s.t. } U(f, P) - L(f, P) < \epsilon\)
    • \(\sum_{i=1}^n \omega_i \Delta x_i < \epsilon\)
  • 黎曼可积充分条件:
    • 函数黎曼可积必有界
    • 闭区间上连续函数必黎曼可积
    • 函数闭区间上第一类间断点个数有限则黎曼可积
  • 性质:
    • 线性性
    • 单调性
    • 若\(f \in \mathcal{R}[a, b]\),则\(|f|\in \mathcal{R}[a, b]\)
    • 区间可加性
    • 黎曼可积函数的乘积黎曼可积
    • 积分中值定理

不定积分
#

  • 定义:变上限积分,原函数
  • Newton-Leibniz公式
  • 积分第一基本定理
    \(F'(x) = \frac{d}{dx} \int_{a}^{x} f(t) \, dt = f(x)\)
  • 积分第二基本定理
    \(F\)是\(f\)区间\(I\)上的一个原函数,则\(F+C\)是区间上所有原函数

两种积分法
#

  • 换元积分法
    • 第一类换元
      \(\int f[\phi(x)] \phi'(x) \, dx = \left[ \int f(u) \, du \right]_{u=\phi(x)}\)
    • 第二类换元
      \(\int f(x) \, dx = (\int f[\psi(t)] \psi'(t) \, dt)_{t=\psi^{-1}(x)}\)
  • 分部积分法
    • 反三角函数\(\approx\)对数函数<幂函数<三角函数\(\approx\)指数函数
    • \(\int u \, dv = uv - \int v \, du\)

反常积分
#

  • 无穷区间上的积分
    • \(\int_{-\infty}^{+\infty} f(x) \, dx = \lim_{a \to -\infty} \int_{a}^{c} f(x) \, dx + \lim_{b \to +\infty} \int_{c}^{b} f(x) \, dx\)
    • P积分
      \(\int_{1}^{+\infty} \frac{1}{x^p} \, dx \quad \begin{cases} \text{\small 收敛} & \text{\small 当 } p > 1 \\ \text{\small 发散} & \text{\small 当 } p \le 1 \end{cases}\)
  • 无界函数积分(瑕积分)
    • \(\int_{a}^{b} f(x) \, dx = \lim_{\epsilon \to 0^+} \int_{a}^{c-\epsilon} f(x) \, dx + \lim_{\eta \to 0^+} \int_{c+\eta}^{b} f(x) \, dx\)
    • P积分
      \(\int_{a}^{b} \frac{dx}{(x-a)^p} \quad \begin{cases} \text{\small 收敛} & \text{\small 当 } p < 1 \\ \text{\small 发散} & \text{\small 当 } p \ge 1 \end{cases}\)
  • 敛散性判别准则:
    • 比较准则(I,II)
    • 绝对收敛准则
    • *狄利克雷判别法
    • *阿贝尔判别法
