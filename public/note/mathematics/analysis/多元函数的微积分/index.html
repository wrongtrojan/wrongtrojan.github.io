n维Euclid空间
#

研究对象所处空间

  1. 向量 \((\alpha_1,\alpha2,...,\alpha_n)\)

  2. 向量加法 \((\alpha_1,\alpha2,...,\alpha_n)+(\beta_1,\beta_2,...,\beta_n)=(\alpha_1+\beta_1,\alpha_2+\beta2,...,\alpha_n+\beta_n)\)

  3. 向量数乘 \(k(\alpha_1,\alpha2,...,\alpha_n)=(k\alpha_1,k\alpha2,...,k\alpha_n)\)

  4. 向量内积 \((\alpha_1,\alpha2,...,\alpha_n)\cdot(\beta_1,\beta_2,...,\beta_n)=\sum_{i=1}^{n}\alpha_i\beta_i\)

点列极限
#

离散的点列的极限

  1. 定义: \(\forall\epsilon>0,\exist N\in N_{+},\forall k>N,\|x_{k}-a\|<\epsilon\)

  2. 点列收敛于\(P_0 \iff\)点列的的各分量组成数列收敛于\(P_0\)的各分量

  3. 唯一性,有界性,线性

  4. Bolzano-Weierstrass定理: 有界点列必有收敛子列

  5. Cauchy收敛原理:
    \(\forall \epsilon>0,\exist N\in N_{+},\forall k>N\)及\(p\in N_{+}\),有\(\|x_{k+p}-x_{k}\|<\epsilon\),则称\(x_{k}\)为Cauchy点列
    点列\(x_{k}\)收敛于\(R^{n}\)中某点\(\iff{x_{k}}\)为Cuchy点列

集合概念
#

对点构成的集合的讨论(以集合A为例)

内部,外部,边界
#

  1. 内点与内部: \(\exist\delta>0,U(P_{0},\delta)\subseteq A\)则称点\(P_0\)是内点,所有内点构成内部,记为\(intA\)

  2. 外点与外部: \(\exist\delta>0,U(P_{0},\delta)\subseteq A^{c}\)则称点\(P_0\)是外点,所有内点构成外部,记为\(extA\)

  3. 边界点与边界: \(\forall\delta>0,\exist P_{1},P_{2}\in U(P_{0},\delta),P_{1}\in A,P_{2}\in A^{c}\)则称点\(P_0\)是边界点,所有边界点构成边界,记为\(\partial A\)

聚点与孤立点
#

  1. 聚点: \(\exist\{x_{k}\},x_k\neq a,有x_k\to a(k\to \infty)\),则称a为A的聚点
    a为A的聚点\(\iff \forall \epsilon>0,\mathring{U}(a,\epsilon)\cap A\neq\emptyset\)

  2. 导集与闭包: 所有聚点构成的集合为导集,记为\(A^{'}\),闭包是A与A的导集的并集,即 \(\overline{A}=A\cup A^{'}\)

  3. 孤立点: \(a\in A,a\notin A^{'}\)

开集与闭集
#

  1. 开集: A中所有的点都是A的内点,即\(A\subseteq intA\)

  2. 闭集: A的所有聚点都属于A,即\(A^{'}\subseteq A\)

  3. Euclid空间中的开集性质

    1. \(\emptyset,R^{n}\)是开集
    2. 任意多开集的并是开集
    3. 有限多个开集的交是开集

  4. Euclid空间中的闭集性质

    1. \(\emptyset,R^{n}\)是闭集
    2. 任意多闭集的交是闭集
    3. 有限多个闭集的并是闭集

区域
#

  1. 有界集/无界集: \(\forall x\in A,\|x\|

  2. 紧集: 有界的闭集

  3. 连通集与凸集:
    A中任意两点x,y能用完全属于A的有限个线段联结,则集合为连通集
    若任意两点直接相连的线段都属于A,则集合为凸集

  4. 开区域与闭区域: 连通的开集是开区域,开区域与其边界的并是闭区域

多元函数的极限与连续性
#

推广一元函数的极限与连续性

  1. 数量值函数与向量值函数:
    n元数量值函数: \(f(X)=f(x_1,x_2,...,x_n)\)
    n元向量值函数: \(Y=F(X),Y=(y_1,y_2,...y_n),F=(f_1,f_2,...f_n)\)

  2. 多元函数的极限(以二元函数为例)
    二重极限: \(\forall\epsilon>0,\exist\delta>0,\forall P_0\in\mathring{U}(P_0,\delta)\cap A,\|f(P_0)-a\|<\epsilon\),记为\(\lim\limits_{(x,y)\to(x_0,y_0)}f(x,y)=a\)
    累次极限: \(\lim\limits_{x\to x_0}\lim\limits_{y\to y_0}f(x,y)\)

  3. 连续性: 若\(f(x,y)\)在\(U((x_0,y_0))\)内有\(\lim\limits_{(x,y)\to(x_0,y_0)}f(x,y)=f(x_0,y_0)\),则\(f(x,y)\)在\((x_0,y_0)\)处连续

  4. 有界闭区域上多元连续函数的性质

    1. 有界性
    2. 最值存在定理
    3. 介质定理
    4. 一致连续性

多元数量值函数的导数与微分
#

全微分与偏导数
#

全微分
#

一元函数微分的直接推广

若\(\Delta u=f(X_0+\Delta X)-f(X_0)=A\Delta x+B\Delta y+o(\rho)\),则称\(A\Delta x+B\Delta y\)是\(f(x,y)\)在\(X_0\)的全微分

偏导数
#

令\(\Delta x\)或\(\Delta y\)为0(以\(\Delta y=0\)为例)

  1. \(f(x_0+\Delta x,y_0)-f(x_0,y_0)=A\Delta x+o(\rho)\)

    \(\lim\limits_{\Delta x\to 0}\frac{f(x_0+\Delta x,y_0)-f(x_0,y_0)}{\Delta x}=A\),记为\(f_x(x_0,y_0)\)

  2. 偏导数的几何意义: \(z=f(x,y)\)与\(y=y_0\)的交线\(z=f(x,y_0)\)在\(x=x_0\)处的斜率

  3. \(\Delta u=f_x\Delta x+f_y\Delta y+o(\rho)\),即\(df=f_xdx+f_ydy\)

可微的条件
#

必要条件
#

(1) \(f\)在\((x_0,y_0)\)连续
(2) \(f\)在\((x_0,y_0)\)的两个偏导均存在

充分条件
#

  1. \(f\)在\((x_0,y_0)\)的两个偏导均连续

  2. \(f_x\)在\(x_0,y_0\)处连续,且\(f_y(x_0,y_0)\)

全微分的有理运算法则
#

与一元函数具有相同形式

  1. \(d(u\pm v)=du\pm dv\)

  2. \(d(uv)=vdu+udv\)

  3. \(d(\frac{u}{v})=\frac{vdu-udv}{v^{2}}\)

复合函数的全微分与偏导
#

链式法则

  1. \(f(u,v),u=u(x,y),v=v(x,y)\)

    1. \(df=(f_1\frac{\partial u}{\partial x}+f_2\frac{\partial v}{\partial x})dx+(f_1\frac{\partial u}{\partial y}+f_2\frac{\partial v}{\partial y})dy\)
    2. \(f_x=f_1\frac{\partial u}{\partial x}+f_2\frac{\partial v}{\partial x},f_y=f_1\frac{\partial u}{\partial y}+f_2\frac{\partial v}{\partial y}\)

  2. \(f(r),r=f(x,y)\)

    1. \(df=f^{'}(r)\frac{\partial r}{\partial x}dx+f^{'}(r)\frac{\partial r}{\partial y}dy\)
    2. \(f_x=f^{'}(r)\frac{\partial r}{\partial x},f_y=f^{'}(r)\frac{\partial r}{\partial y}\)

  3. \(f(x,y),x=x(t),y=y(t)\)

    1. \(df=\frac{\partial f}{\partial x}x^{'}(t)+\frac{\partial f}{\partial y}y^{'}(t)\)

梯度与方向函数
#

描述函数在某方向上的变化

  1. 方向导数: \(\lim\limits_{t \to 0}\frac{f(X_0+te_1)-f(X_0)}{t}\)

\(\lim\limits_{t \to 0}\frac{f(x_0+t\cos\theta,y_0+t\sin\theta)-f(x_0,y_0)}{t}=\lim\limits_{t \to 0}\frac{f_xt\cos\theta+f_yt\sin\theta+o(\rho)}{t}=f_x\cos\theta+f_y\sin\theta\)

  1. 梯度

向量方向描述函数增长最快方向,向量大小描述函数增长最快速度

\(\nabla f=(f_x,f_y)=(\frac{\partial f}{\partial x},\frac{\partial f}{\partial y})\) \(\nabla=(\frac{\partial}{\partial x},\frac{\partial}{\partial y})\)

高阶偏导数和全微分
#

  1. 高阶偏导数

    • \(\frac{\partial^2z}{(\partial x)^2}\),\(\frac{\partial^2z}{(\partial y)^2}\)
    • \(\frac{\partial^2z}{(\partial x)(\partial y)}=\frac{\partial^2z}{(\partial y)(\partial x)}\),当\(f_{xy},f_{yx}\)在点\(P\)均连续时

  2. 高阶全微分

    • \(du=\langle\nabla,\Delta X\rangle=\frac{\partial f}{\partial x}dx+\frac{\partial f}{\partial y}dy\)
    • \(d^2u=\frac{\partial^2u}{(\partial x)^2}d^2x+2\frac{\partial^2u}{(\partial x)(\partial y)}dxdy+\frac{\partial^2u}{(\partial y)^2}d^2y=(\frac{\partial}{\partial x}dx+\frac{\partial}{\partial y}dy)^2u\)

隐函数微分
#

  1. 一元隐函数

  • 隐函数存在唯一定理: (1)\(F\)在\((x_0,y_0)\)某一领域连续
    (2)\(F(x_0,y_0)=0\)
    (3)\(F_{y}\)在\((x_0,y_0)\)附近连续,且\(F_y(x_0,y_0)\neq0\)
    (4)\(F_{x}\)在\((x_0,y_0)\)附近连续

  • 若满足(1)~(3),则隐函数存在且唯一
    若还满足(4),则隐函数可微,且有\(\frac{dy}{dx}=-\frac{F_x}{F_y}\)

  1. 多元隐函数(以二元为例)
  • \(F(x,y,z)=0\)
    \(\frac{\partial z}{\partial x}=-\frac{F_x}{F_z},\frac{\partial z}{\partial y}=-\frac{F_y}{F_z}\)

关于\(\partial\)算符:\(\frac{\partial}{\partial x}\)
#

  1. 线性性质: \(\frac{\partial}{\partial x}(af+bg)=a\frac{\partial f}{\partial x}+b\frac{\partial g}{\partial x}\)

  2. 形式代数运算

    1. \(\frac{\partial^2z}{(\partial x)(\partial y)}=(\frac{\partial }{\partial x})(\frac{\partial }{\partial y})u\)
    2. \(d^2u=(\frac{\partial}{\partial x}dx+\frac{\partial}{\partial y}dy)^2u\)

  3. 坐标变换(以\((x,y)\to(\zeta,\eta)\)为例)

    • \(du=\frac{\partial u}{\partial x}dx+\frac{\partial u}{\partial y}dy=\frac{\partial u}{\partial \zeta}d\zeta+\frac{\partial u}{\partial \eta}d\eta\)…(1)
    • \(d\zeta=\frac{\partial \zeta}{\partial x}dx+\frac{\partial \zeta}{\partial y}dy\),\(d\eta=\frac{\partial \eta}{\partial x}dx+\frac{\partial \eta}{\partial y}dy\)…(3)
    • 将(2)带入(1),有\(\frac{\partial }{\partial x}=\frac{\partial }{\partial \zeta}\cdot\frac{\partial \zeta}{\partial x}+\frac{\partial }{\partial \eta}\cdot\frac{\partial \eta}{\partial x}\),\(\frac{\partial }{\partial y}=\frac{\partial }{\partial \zeta}\cdot\frac{\partial \zeta}{\partial y}+\frac{\partial }{\partial \eta}\cdot\frac{\partial \eta}{\partial y}\)

多元函数的Taylor公式与极值
#

多元函数的Taylor公式
#

由一元函数Taylor公式推广

  • 一阶Lagrange余项:
$$f(X_0+\Delta X)=f(X_0)+\langle\nabla f \Delta X\rangle+R_1$$

其中

$$R_1=\frac{1}{2!}\Delta X^TH_f(X_0+\theta\Delta X)\Delta X$$

注:

$$H_f=\begin{bmatrix} f_{xx} \ f_{xy} \\ f_{xy} \ f_{yy}\end{bmatrix}$$
  • 二阶Peano余项:
$$f(X_0+\Delta X)=f(X_0)+\langle\nabla f \Delta X\rangle+\frac{1}{2!}\Delta X^TH_f(X_0)\Delta X+o\|\Delta X\|^2$$

无约束极值与最值
#

  1. 无约束极值

  • 定义: \(\forall X\in U(X_0),f(X)\leq f(X_0)(f(X)\geq f(X_0))\)恒成立,则\(f\)在\(X_0\)处取到无约束极大值(极小值)

  • 必要条件:

    • 函数\(f\)在\(X_0\)的一阶偏导数存在,若\(f\)在\(X_0\)取到极值,则\(\nabla f(X_0)=0\)
    • 证明:\(\forall X_i\),考虑\(\varphi (t)=f(x_1,x_2,...,x_i+th,...x_n)\),若\(f\)在\(X_0\)取到极值,则\(\varphi (t)\)在\(t=0\)取到极值,则有\(\varphi^{'}(0)=0\),即\(f_{x_i}(X_0)=0\)
    • 若\(\nabla f(X_0)=0\),\(X_0\)称为\(f\)的驻点

  • 充分条件

    • \(f\in C^{(2)}(U(X_0)),\nabla f(X_0)=0\),若\(H_f(X_0)\)正定/负定/不定/半定,则\(f(X_0)\)为\(f\)的极小值/为\(f\)的极大值/不为极值/需要进一步讨论
  1. 最值

  • 对于有界闭域,分别求出驻点,偏导数不存在点,边界点处函数值作比较

  • 对于非闭域,取边界附近(-\(\delta\))处的点做函数值大小判断

有约束极值
#

  • 定义: 对目标函数在限制条件求极值的问题

  • Lagrange乘数法
