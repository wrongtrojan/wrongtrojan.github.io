PDF 借助浏览器原生 Viewer 展示方案（完整可复用）
#

这份文档总结当前项目里已经跑通的方案：
在 Hugo 页面中使用浏览器原生 PDF viewer（iframe），并在外层做 UI 控制、翻页、页数探测和加载遮掩。

1. 方案目标
#

  • 使用浏览器内置 PDF 渲染能力，不引入完整自绘 Viewer
  • 页面内按钮手动翻页（上一页/下一页）
  • 支持自动探测总页数（失败时也不锁死翻页）
  • 减少切页时内置 viewer 的抖动和闪烁
  • 保持 Dracula 风格的加载遮罩视觉

2. 目录结构（当前实现）
# 

layouts/shortcodes/pdfviewer.html
assets/css/custom.css
static/assets/DataPath.pdf
static/lib/pdfjs/pdf.min.js
static/lib/pdfjs/pdf.worker.min.js
content/note/ComputerScience/CSAPP/CPU.md

说明：

  • static/assets/*.pdf 是 PDF 静态资源
  • static/lib/pdfjs/* 仅用于读取总页数（pdfjsLib.getDocument），不用于主渲染

4. Shortcode 完整源码（真实可用）
#

文件：layouts/shortcodes/pdfviewer.html

<div
  class="pdf-lite"
  data-url="{{ .Get `url` | default `/assets/DataPath.pdf` }}"
  data-page="{{ .Get `page` | default `1` }}"
  data-max="{{ .Get `max` | default `0` }}"
>
  <div class="pdf-stage">
    <div class="pdf-toolbar">
      <button class="pdf-prev" type="button" aria-label="Previous page">◀</button>
      <span class="pdf-page-label">1 / --</span>
      <button class="pdf-next" type="button" aria-label="Next page">▶</button>
    </div>
    <iframe
      class="pdf-frame"
      title="PDF"
      src="{{ .Get `url` | default `/assets/DataPath.pdf` }}#page={{ .Get `page` | default `1` }}&view=Fit&toolbar=0&navpanes=0&scrollbar=0"
    ></iframe>
    <div class="pdf-loading-mask" aria-hidden="true"></div>
    <div class="pdf-wheel-mask" aria-hidden="true"></div>
  </div>
</div>

<script>
(() => {
  const root = document.currentScript?.previousElementSibling;
  if (!root || !root.classList.contains('pdf-lite')) return;

  const url = root.dataset.url;
  let max = Number(root.dataset.max || 0);
  let page = Number(root.dataset.page || 1);

  const frame = root.querySelector('.pdf-frame');
  const stage = root.querySelector('.pdf-stage');
  const prev = root.querySelector('.pdf-prev');
  const next = root.querySelector('.pdf-next');
  const label = root.querySelector('.pdf-page-label');
  const mask = root.querySelector('.pdf-wheel-mask');
  const loadingMask = root.querySelector('.pdf-loading-mask');
  let loadingTimer = null;
  let loadingStartedAt = 0;

  const setLoading = (loading) => {
    if (!loadingMask) return;
    if (loading) loadingStartedAt = Date.now();
    loadingMask.classList.toggle('is-active', loading);
    stage?.classList.toggle('is-loading', loading);
  };

  const clearLoadingWithMinDuration = () => {
    const minVisibleMs = 420;
    const elapsed = Date.now() - loadingStartedAt;
    const wait = Math.max(0, minVisibleMs - elapsed);
    if (loadingTimer) clearTimeout(loadingTimer);
    loadingTimer = setTimeout(() => {
      setLoading(false);
      loadingTimer = null;
    }, wait);
  };

  frame?.addEventListener('load', () => {
    clearLoadingWithMinDuration();
  });

  const ensurePdfJs = async () => {
    if (window.pdfjsLib) return window.pdfjsLib;
    await new Promise((resolve, reject) => {
      const script = document.createElement('script');
      script.src = '/lib/pdfjs/pdf.min.js';
      script.async = true;
      script.onload = resolve;
      script.onerror = reject;
      document.head.appendChild(script);
    });
    if (!window.pdfjsLib) throw new Error('pdfjs not loaded');
    window.pdfjsLib.GlobalWorkerOptions.workerSrc =
      '/lib/pdfjs/pdf.worker.min.js';
    return window.pdfjsLib;
  };

  const resolveTotalPages = async () => {
    if (max > 0) return;
    try {
      const pdfjsLib = await ensurePdfJs();
      const doc = await pdfjsLib.getDocument(url).promise;
      max = doc.numPages || 1;
      render();
    } catch (_) {
      // Keep unknown total when auto-detection fails; don't lock paging to 1.
      max = 0;
      render();
    }
  };

  const render = (showLoading = true) => {
    const safeMax = max > 0 ? max : null;
    if (page < 1) page = 1;
    if (safeMax && page > safeMax) page = safeMax;
    const sep = url.includes('?') ? '&' : '?';
    const nextSrc = `${url}${sep}_pg=${page}#page=${page}&view=Fit&toolbar=0&navpanes=0&scrollbar=0`;
    if (showLoading) {
      setLoading(true);
      if (loadingTimer) clearTimeout(loadingTimer);
      loadingTimer = setTimeout(() => {
        setLoading(false);
        loadingTimer = null;
      }, 1300);
    }
    requestAnimationFrame(() => {
      try {
        if (frame.contentWindow) {
          frame.contentWindow.location.replace(nextSrc);
        } else {
          frame.src = nextSrc;
        }
      } catch (_) {
        frame.src = nextSrc;
      }
    });
    label.textContent = safeMax ? `${page} / ${safeMax}` : `${page} / --`;
    prev.disabled = page <= 1;
    next.disabled = safeMax ? page >= safeMax : false;
  };

  prev?.addEventListener('click', () => {
    if (page > 1) {
      page--;
      render();
    }
  });

  next?.addEventListener('click', () => {
    if (max <= 0 || page < max) {
      page++;
      render();
    }
  });

  // Let page-level scrolling work while hovering over the PDF area.
  mask?.addEventListener('pointerdown', (e) => e.preventDefault());

  render(false);
  resolveTotalPages();
})();
</script>

6. 关键机制说明（为什么这样做）
#

6.1 原生 viewer 翻页触发
#

  • 仅改 hash（#page=n）在部分浏览器不稳定
  • 使用 ?_pg=n#page=n... 让 URL 每次变化，触发可靠刷新
  • 再配合 location.replace/frame.src 回退，提升兼容性

6.2 自动总页数
#

  • 使用本地 pdf.js 读取 numPages
  • 若失败，不锁死为 1 页，而是显示 --，并允许继续翻页

6.3 切页遮掩
#

  • is-loading 时降低 iframe 可见度并轻微模糊，掩盖内置 viewer 的缩放抖动
  • 叠加 Dracula 风格网格和发光点，形成主题化淡入层

6.4 滚动条遮掩
#

  • 通过容器 overflow-x: hidden/clip 抑制横向闪现
  • 通过 ::before/::after 右侧与底部遮罩条隐藏内置滚动条区域

7. 可复用步骤（新页面接入）
#

  1. 把 PDF 放进 static/assets/
  2. 在内容页写：
    • 1 / --
  3. 确保以下文件存在：
    • layouts/shortcodes/pdfviewer.html
    • assets/css/custom.css（含 PDF 样式块）
    • static/lib/pdfjs/pdf.min.js
    • static/lib/pdfjs/pdf.worker.min.js
  4. 重新构建 Hugo 站点

8. 已知边界
#

  • 这是“原生 viewer + 外层增强”，不是完全自绘 PDF viewer
  • 各浏览器对 PDF 内核行为仍有细微差异
  • 若追求完全可控（无重载抖动、精确事件控制），最终仍建议迁移到完整 pdf.js 渲染页方案

9. 当前版本建议保留参数
#

  • view=Fit：更接近整页显示
  • toolbar=0&navpanes=0&scrollbar=0：减少原生 UI 干扰
  • 初始 render(false)：全局首次刷新不触发 loading 淡入

这套配置是当前项目里“视觉和稳定性折中”最好的版本。
